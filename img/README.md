# 画像フォルダ

このフォルダには以下の画像ファイルを配置してください：

## 必要な画像ファイル

1. **hometop1.png** (300x300px推奨)
   - 学生としてのあなたの写真
   - 大学や研究室での写真など

2. **hometop2.png** (300x300px推奨)
   - ロボット研究に関連する写真
   - ロボットの写真や競技会の様子など

3. **hometop3.png** (300x300px推奨)
   - ソフトウェア開発に関連する写真
   - プログラミングの様子やプロジェクトのスクリーンショットなど

## 画像の仕様

- **サイズ**: 300x300px（正方形）推奨
- **フォーマット**: JPG、PNG、WebP対応
- **容量**: 各ファイル1MB以下推奨
- **品質**: 高画質でクリアな画像

## 注意事項

- 画像ファイルが存在しない場合は、自動的にプレースホルダー画像が表示されます
- ファイル名は上記の通り正確に設定してください
- 画像は円形にクロップされて表示されます

## ファイル配置例

```
img/
├── hometop1.png
├── hometop2.png
└── hometop3.png
```
